# Snake

A pure front-end Snake game with a bold, high-contrast vibe. Built for quick play, crisp controls, and a clean visual identity.

## Run

Open `index.html` in a browser.

## Controls

- Arrow keys / WASD: move
- Space: start / pause
- On-screen buttons: mobile control

## Notes

- Best score is stored in `localStorage`.
- Speed slider adjusts the tick rate.
